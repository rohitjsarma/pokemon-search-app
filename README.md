# pokemon-search-app
🧩 Pokémon Search App A simple and responsive Pokémon search application built using Next.js (App Router) and Tailwind CSS. It allows users to filter Pokémon by type and search by name, and provides detailed information on each Pokémon on a separate page.  🔍 Features ✅ Search Pokémon by name  ✅ Filter Pokémon by type (Fire, Water, Grass, etc.)
