## 🧩 Pokémon Search App

A simple and responsive Pokémon search application built using **Next.js (App Router)** and **Tailwind CSS**.  
It allows users to **filter Pokémon by type** and **search by name**, and provides detailed information on each Pokémon on a separate page.

### 🔍 Features
- ✅ Search Pokémon by **name**
- ✅ Filter Pokémon by **type** (Fire, Water, Grass, etc.)
- ✅ Click on a Pokémon card to view **detailed stats**
- ✅ **Breadcrumb navigation** on the detail page
- ✅ **Responsive UI** for mobile and desktop
- ✅ Built using **Next.js App Router**, **Tailwind CSS**
- ✅ Uses **PokeAPI** for real-time Pokémon data
- ✅ Clean and maintainable code with **custom hooks** and **server/client components**

### 🧪 Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Data**: [PokeAPI](https://pokeapi.co/)
- **Routing**: Dynamic Routes with Breadcrumbs
- **Rendering**: Server-Side Rendering (SSR) + Client Components
- **Hooks**: Custom React hooks for data fetching and filtering
